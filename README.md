# McDonald's Macro Checker

A web application built with Bun for checking nutritional macros at McDonald's.

## Tech Stack

- **Runtime**: Bun
- **Deployment**: Vercel

## Development

Install dependencies:
```bash
bun install
```

Run development server:
```bash
bun run dev
```

Build for production:
```bash
bun run build
```

## Deployment

Deploy to Vercel:
```bash
vercel --prod --yes
```

## Type Checking

Run type check without building:
```bash
bunx tsc --noEmit
```
